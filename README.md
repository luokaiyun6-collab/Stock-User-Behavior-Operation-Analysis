# 股票投资用户行为分析与运营策略优化（Excel项目）

## 项目概述
基于沪深300指数2023-2026年历史数据，使用Excel完成用户行为分层分析与运营策略模拟。  
核心目标：发现不同风险偏好用户的留存/活跃规律，提出分层运营建议（适用于互联网金融App、电商私域、用户运营岗位）。

## 主要成果
- 数据处理：清洗历史K线数据，计算每日收益率、20日均线、波动率等指标
- 用户分层：波动率 > 全样本平均 → 高风险用户，其余 → 低风险用户
- 关键发现：
  - 低风险用户（占比约65%）：平均每日收益率 +0.019%（稳定正收益），波动率较低（0.75%）
  - 高风险用户（占比约32%）：平均每日收益率 -0.094%（负收益），波动剧烈（1.36%），易流失
  - 总体：小幅负收益（-0.019%），波动率0.95%
- 运营策略建议：分层维护（高风险：风险教育+预警推送；低风险：VIP权益+专属活动），模拟活跃度提升15%以上
